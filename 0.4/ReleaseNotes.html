Tempora 0.4

  • New — Timer presets. Save a reusable timer setup — a title, optionally tied to a project — once, then start from it in a click. Presets live under their project in the sidebar, double as filters that narrow the list to just that work, accept drops (drag an entry, a running timer, or tracked app usage onto a preset to make it match), and appear in the New Timer popover in the toolbar and menu bar.
  • New — You can now jot notes on a timer while creating it in the New Timer form.
  • Fixed — discarding a running timer from the menu-bar popover now works. Previously the confirmation dismissed the whole popover without discarding.
  • Fixed — browser tab tracking now honours the per-browser opt-in, so a browser you have not enabled is no longer read.
