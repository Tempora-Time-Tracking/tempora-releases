Tempora 0.4.1

  • New — Edit a running timer. Open the actions menu on the active timer and choose Edit to change its title, project, notes, or start time without stopping it.
  • New — Open Tempora directly from the menu-bar timer popover, above Settings.
  • New — Delete a project or a timer preset from its edit popover, so you no longer have to reach for the sidebar's context menu.
  • Improved — a custom start time for a new or edited timer can no longer be set in the future.
