Tempora 0.4.4

  • New — Create a ticket straight from the sidebar. The ➕ menu now has a "New Ticket" action, so you can add a ticket to a project without starting a timer first.
  • New — Tickets are included in JSON export and import. Each project's tickets travel with it, and importing a backup restores them (skipping any that already exist under the same project).
  • Changed — The Settings window has a cleaner layout.
  • Improved — Various technical improvements under the hood.
